publicVariablesGet

Get public variables

URL

https://localhost/api/public/variables

HTTP Method

GET

Response Type

Parameters

publicVariablesSearchSearchGet

Get top 5 PUBLIC variables with the most correlations containing the entered search characters. For example, search for 'mood' as an effect. Since 'Overall Mood' has a lot of correlations with other variables, it should be in the autocomplete list.

URL

https://localhost/api/public/variables/search/{search}

HTTP Method

GET

Response Type

Parameters

    Search query

    Allows us to specify which column in the `correlations` table will be searched. Choices are effect or cause.

GET

Response Type

Parameters

variablesVariableNameGet

Get all of the settings and information about a variable by name

URL

https://localhost/api/variables/{variableName}

HTTP Method

GET

Response Type

Parameters

    Variable name

    categoryName` parameter allows us to filter the variables so that only ones from the specified category are returned.
