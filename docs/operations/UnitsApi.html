This is the API

v1UnitCategoriesGet

Get a list of the categories of measurement units such as 'Distance', 'Duration', 'Energy', 'Frequency', 'Miscellany', 'Pressure', 'Proportion', 'Rating', 'Temperature', 'Volume', and 'Weight'.

URL

https://localhost/api/v1/unitCategories

HTTP Method

GET

Response Type

UnitCategory

Parameters

v1UnitsGet

Get all available units

URL

https://localhost/api/v1/units

HTTP Method

GET

Response Type

List

Parameters

  • Query: unitName String

    Unit name

  • Query: abbreviatedUnitName String

    Restrict the results to a specific unit by providing the unit abbreviation.

  • Query: categoryName String

    Restrict the results to a specific unit category by providing the unit category name.

v1UnitsVariableGet

Get a list of all possible units to use for a given variable

URL

https://localhost/api/v1/unitsVariable

HTTP Method

GET

Response Type

List

Parameters

  • Query: unitName String

    Name of Unit you want to retrieve

  • Query: abbreviatedUnitName String

    Abbreviated Unit Name of the unit you want

  • Query: categoryName String

    Name of the category you want units for

  • Query: variable String

    Name of the variable you want units for
