v1MeasurementSourcesGet

HTTP Method

GET

Response Type

MeasurementSource

Parameters

v1MeasurementSourcesPost

Add a life-tracking app or device to the QuantiModo list of data sources.

URL

https://localhost/api/v1/measurementSources

HTTP Method

POST

Response Type

Parameters

  • Body: name MeasurementSource

    An array of names of data sources you want to add.

v1MeasurementsGet

Measurements are any value that can be recorded like daily steps, a mood rating, or apples eaten. <br>Supported filter parameters:<br><ul><li><b>value</b> - Value of measurement</li><li><b>lastUpdated</b> - The time that this measurement was created or last updated in the UTC format \"YYYY-MM-DDThh:mm:ss\"</li></ul><br>

URL

https://localhost/api/v1/measurements

HTTP Method

GET

Response Type

Measurement

Parameters

  • Query: variableName String

    Name of the variable you want measurements for

  • Query: unit String

    The unit your want the measurements in

  • Query: startTime String

    The lower limit of measurements returned (Epoch)

  • Query: endTime String

    The upper limit of measurements returned (Epoch)

  • Query: groupingWidth Integer

    The time (in seconds) over which measurements are grouped together

  • Query: groupingTimezone String

    The time (in seconds) over which measurements are grouped together

  • Query: limit Integer

    The LIMIT is used to limit the number of results returned. So if you have 1000 results, but only want to the first 10, you would set this to 10 and offset to 0.

  • Query: offset Integer

    Now suppose you wanted to show results 11-20. You'd set the offset to 10 and the limit to 10.

  • Query: sort Integer

    Sort by given field. If the field is prefixed with `-, it will sort in descending order.

v1MeasurementsPost

You can submit or update multiple measurements in a \"measurements\" sub-array. If the variable these measurements correspond to does not already exist in the database, it will be automatically added. The request body should look something like [{\"measurements\":[{\"timestamp\":1406419860,\"value\":\"1\",\"note\":\"I am a note about back pain.\"},{\"timestamp\":1406519865,\"value\":\"3\",\"note\":\"I am another note about back pain.\"}],\"name\":\"Back Pain\",\"source\":\"QuantiModo\",\"category\":\"Symptoms\",\"combinationOperation\":\"MEAN\",\"unit\":\"/5\"}]

URL

https://localhost/api/v1/measurements

HTTP Method

POST

Response Type

Parameters

  • Body: measurements MeasurementSet

    An array of measurements you want to insert.

v1MeasurementsRangeGet

Get Unix time-stamp (epoch time) of the user's first and last measurements taken.

URL

https://localhost/api/v1/measurementsRange

HTTP Method

GET

Response Type

MeasurementRange

Parameters

  • Query: sources String

    Enter source name to limit to specific source (varchar)

  • Query: user Integer

    If not specified, uses currently logged in user (bigint)
