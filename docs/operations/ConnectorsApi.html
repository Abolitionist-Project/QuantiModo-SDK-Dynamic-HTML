This is the API

connectorsListGet

Returns a list of all available connectors. A connector pulls data from other data providers using their API or a screenscraper.

URL

https://localhost/api/connectors/list

HTTP Method

GET

Response Type

List

Parameters

connectorsConnectorConnectGet

Attempt to obtain a token from the data provider, store it in the database. With this, the connector to continue to obtain new user data until the token is revoked.

URL

https://localhost/api/connectors/{connector}/connect

HTTP Method

GET

Response Type

Parameters

  • Path: connector String

    Lowercase system name of the source application or device. Get a list of available connectors from the /connectors/list endpoint.

connectorsConnectorConnectInstructionsGet

Returns instructions that describe what parameters and endpoint to use to connect to the given data provider.

URL

https://localhost/api/connectors/{connector}/connectInstructions

HTTP Method

GET

Response Type

Parameters

  • Path: connector String

    Lowercase system name of the source application or device. Get a list of available connectors from the /connectors/list endpoint.

  • Query: url String

    URL which should be used to enable the connector

  • Query: parameters List(String)

    Array of Parameters for the request to enable connector

  • Query: usePopup Boolean

    Should use popup when enabling connector

connectorsConnectorConnectParameterGet

Returns instructions that describe what parameters and endpoint to use to connect to the given data provider.

URL

https://localhost/api/connectors/{connector}/connectParameter

HTTP Method

GET

Response Type

Parameters

  • Path: connector String

    Lowercase system name of the source application or device. Get a list of available connectors from the /connectors/list endpoint.

  • Query: displayName String

    Name of the parameter that is user visible in the form

  • Query: key String

    Name of the property that the user has to enter such as username or password Connector (used in HTTP request) TODO What's a connector key?

  • Query: usePopup Boolean

    Should use popup when enabling connector

  • Query: type String

    Type of input field such as those found here http://www.w3schools.com/tags/tag_input.asp

  • Query: placeholder String

    Placeholder hint value for the parameter input tag

  • Query: defaultValue String

    Default parameter value

connectorsConnectorDisconnectGet

The disconnect method deletes any stored tokens or connection information from the connectors database.

URL

https://localhost/api/connectors/{connector}/disconnect

HTTP Method

GET

Response Type

Parameters

  • Path: connector String

    Lowercase system name of the source application or device. Get a list of available connectors from the /connectors/list endpoint.

connectorsConnectorInfoGet

Returns information about the connector such as the connector id, whether or not is connected for this user (i.e. we have a token or credentials), and its update history for the user.

URL

https://localhost/api/connectors/{connector}/info

HTTP Method

GET

Response Type

Parameters

  • Path: connector String

    Lowercase system name of the source application or device. Get a list of available connectors from the /connectors/list endpoint.

connectorsConnectorUpdateGet

The update method tells the QM Connector Framework to check with the data provider (such as Fitbit or MyFitnessPal) and retrieve any new measurements available.

URL

https://localhost/api/connectors/{connector}/update

HTTP Method

GET

Response Type

Parameters
