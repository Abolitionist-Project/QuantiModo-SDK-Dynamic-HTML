This is the API

correlationsGet

Get correlations

URL

https://localhost/api/correlations

HTTP Method

GET

Response Type

List

Parameters

    ORIGINAL variable name of the effect variable for which the user desires correlations

    ORIGINAL variable name of the cause variable for which the user desires correlations

publicCorrelationsSearchSearchGet

Returns the average correlations from all users for all public variables that contain the characters in the search query.

URL

https://localhost/api/public/correlations/search/{search}

HTTP Method

GET

Response Type

List

Parameters

    Search query

    Specifies whehter to return the effects or causes of the searched variable.
