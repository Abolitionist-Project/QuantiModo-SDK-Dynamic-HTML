This is the API

correlationsGet

Get correlations

URL

https://localhost/api/correlations

HTTP Method

GET

Response Type

List

Parameters

  • Query: effect String

    ORIGINAL variable name of the effect variable for which the user desires correlations

  • Query: cause String

    ORIGINAL variable name of the cause variable for which the user desires correlations

publicCorrelationsSearchSearchGet

Returns the average correlations from all users for all public variables that contain the characters in the search query. Returns average of all users public variable correlations with a specified cause or effect.

URL

https://localhost/api/public/correlations/search/{search}

HTTP Method

GET

Response Type

List

Parameters

  • Path: search String

    Name of the variable that you want to know the causes or effects of.

  • Query: effectOrCause String

    Specifies whether to return the effects or causes of the searched variable.

v1OrganizationsOrganizationIdUsersUserIdVariablesVariableNameCausesGet

Returns average of all correlations and votes for all user cause variables for a given effect. If parameter \"include_public\" is used, it also returns public correlations. User correlation overwrites or supersedes public correlation.

URL

https://localhost/api/v1/organizations/{organizationId}/users/{userId}/variables/{variableName}/causes

HTTP Method

GET

Response Type

List

Parameters

  • Path: organizationId Integer

    Organization ID

  • Path: userId Integer

    User id

  • Path: variableName String

    Effect variable name

  • Query: organizationToken String

    Organization access token

  • Query: includePublic String

    Include bublic correlations, Can be \"1\" or empty.
