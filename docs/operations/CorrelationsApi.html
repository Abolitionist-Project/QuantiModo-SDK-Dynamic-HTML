v1AggregatedCorrelationsGet

Get correlations based on the anonymized aggregate data from all QuantiModo users.

URL

https://app.quantimo.do/api/v1/aggregatedCorrelations

HTTP Method

GET

Response Type

List

Parameters

  • Query: access_token String

    User's OAuth2 access token

  • Query: effect String

    ORIGINAL variable name of the effect variable for which the user desires correlations

  • Query: cause String

    ORIGINAL variable name of the cause variable for which the user desires correlations

  • Query: correlationCoefficient String

    Pearson correlation coefficient between cause and effect after lagging by onset delay and grouping by duration of action

  • Query: onsetDelay String

    The number of seconds which pass following a cause measurement before an effect would likely be observed.

  • Query: durationOfAction String

    The time in seconds over which the cause would be expected to exert a measurable effect. We have selected a default value for each variable. This default value may be replaced by a user specified by adjusting their variable user settings.

  • Query: lastUpdated String

    The time that this measurement was last updated in the UTC format \"YYYY-MM-DDThh:mm:ss\"

  • Query: limit Integer

    The LIMIT is used to limit the number of results returned. So if you have 1000 results, but only want to the first 10, you would set this to 10 and offset to 0.

  • Query: offset Integer

    Now suppose you wanted to show results 11-20. You'd set the offset to 10 and the limit to 10.

  • Query: sort Integer

    Sort by given field. If the field is prefixed with `-, it will sort in descending order.

Returns average of all correlations and votes for all user cause variables for a given cause. If parameter \"include_public\" is used, it also returns public correlations. User correlation overwrites or supersedes public correlation.

URL

https://app.quantimo.do/api/v1/organizations/{organizationId}/users/{userId}/variables/{variableName}/causes

HTTP Method

GET

Response Type

List

Parameters

  • Path: organizationId Integer

    Organization ID

  • Path: userId Integer

    User id

  • Path: variableName String

    Effect variable name

  • Query: organization_token String

    Organization access token

  • Query: access_token String

    User's OAuth2 access token

  • Query: includePublic String

    Include public correlations, Can be \"1\" or empty.

v1PublicCorrelationsSearchSearchGet

Returns the average correlations from all users for all public variables that contain the characters in the search query. Returns average of all users public variable correlations with a specified cause or effect.

URL

https://app.quantimo.do/api/v1/public/correlations/search/{search}

HTTP Method

GET

Response Type

List

Parameters

  • Path: search String

    Name of the variable that you want to know the causes or effects of.

  • Query: effectOrCause String

    Setting this to effect indicates that the searched variable is the effect and that the causes of this variable should be returned. cause indicates that the searched variable is the cause and the effects should be returned.

  • Query: access_token String

    User's OAuth2 access token

v1VotesPost

This is to enable users to indicate their opinion on the plausibility of a causal relationship between a treatment and outcome. QuantiModo incorporates crowd-sourced plausibility estimations into their algorithm. This is done allowing user to indicate their view of the plausibility of each relationship with thumbs up/down buttons placed next to each prediction.

URL

https://app.quantimo.do/api/v1/votes

HTTP Method

POST

Response Type

CommonResponse

Parameters

  • Body: body PostVote(PostVote)

    Contains the cause variable, effect variable, and vote value.

  • Query: access_token String

    User's OAuth2 access token
