This is the API

administrationCacheClearGet

Clears the memcached cache for the units and some other things.

URL

https://localhost/api/administration/cache/clear

HTTP Method

GET

Response Type

Parameters

administrationCorrelationsUpdateGet

Recalculate correlations for a variable

URL

https://localhost/api/administration/correlations/update

HTTP Method

GET

Response Type

Parameters

    Original name of variable for which correlations are to be calculated (varchar)

    ID for user for whom the correlations should be recalculated.

administrationCorrelationsUpdateAllUsersGet

Recalculates correlations for all variables, for all users.

URL

https://localhost/api/administration/correlations/updateAllUsers

HTTP Method

GET

Response Type

Parameters

administrationCorrelationsUpdateAllVariablesGet

Updates the correlations for all variables for a given user.

URL

https://localhost/api/administration/correlations/updateAllVariables

HTTP Method

GET

Response Type

Parameters

    ID for user for whom the correlations should be recalculated. If empty, uses currently logged in user.
