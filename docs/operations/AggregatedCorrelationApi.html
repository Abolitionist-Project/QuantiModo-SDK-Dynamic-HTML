This is the API

aggregatedCorrelationsGet

Get all AggregatedCorrelations

URL

https://app.quantimo.do/api/v2/aggregatedCorrelations

HTTP Method

GET

Response Type

inline_response_200

Parameters

  • Query: access_token String

    User's OAuth2 access token

  • Query: correlation BigDecimal

    Pearson correlation coefficient between cause and effect measurements

  • Query: cause_id Integer

    Variable ID of the predictor variable for which the user desires correlations

  • Query: effect_id Integer

    Variable ID of the outcome variable for which the user desires correlations

  • Query: onset_delay Integer

    User estimated (or default number of seconds) after cause measurement before a perceivable effect is observed

  • Query: duration_of_action Integer

    Number of seconds over which the predictor variable event is expected to produce a perceivable effect following the onset delay

  • Query: number_of_pairs Integer

    Number of predictor/outcome data points used in the analysis

  • Query: value_predicting_high_outcome BigDecimal

    Predictor daily aggregated measurement value that predicts an above average effect measurement value (in default unit for predictor variable)

  • Query: value_predicting_low_outcome BigDecimal

    Predictor daily aggregated measurement value that predicts a below average effect measurement value (in default unit for outcome variable)

  • Query: optimal_pearson_product BigDecimal

    Optimal Pearson Product

  • Query: number_of_users Integer

    Number of users whose data was used in this aggregation

  • Query: number_of_correlations Integer

    Number of correlational analyses used in this aggregation

  • Query: statistical_significance BigDecimal

    A function of the effect size and sample size

  • Query: cause_unit String

    Abbreviated unit name for the predictor variable

  • Query: cause_unit_id Integer

    Unit ID for the predictor variable

  • Query: cause_changes Integer

    Number of times that the predictor time series changes

  • Query: effect_changes Integer

    Number of times that the predictor time series changes

  • Query: aggregate_qm_score BigDecimal

    Aggregated QM Score which is directly proportional with the relevance of each predictor or outcome

  • Query: created_at String

    Date at which the analysis was first performed

  • Query: updated_at String

    Date at which the analysis was last updated

  • Query: status String

    Indicates whether an analysis is up to date (UPDATED), needs to be updated (WAITING), or had an error (ERROR)

  • Query: error_message String

    Message describing any problems encountered during the analysis

  • Query: last_successful_update_time String

    Last Successful update time

  • Query: reverse_pearson_correlation_coefficient BigDecimal

    Correlation when cause and effect are reversed. For any causal relationship, the forward correlation should exceed the reverse correlation

  • Query: predictive_pearson_correlation_coefficient BigDecimal

    Predictive Pearson Correlation Coefficient

  • Query: limit Integer

    Limit the number of results returned

  • Query: offset Integer

    Records from give Offset

  • Query: sort String

    Sort records by given field
