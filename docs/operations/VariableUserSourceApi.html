This is the API

variableUserSourcesGet

Get all VariableUserSources

URL

https://app.quantimo.do/api/v2/variableUserSources

HTTP Method

GET

Response Type

inline_response_200_10

Parameters

  • Query: access_token String

    User's OAuth2 access token

  • Query: variable_id Integer

    ID of variable

  • Query: user_id Integer

    ID of User

  • Query: timestamp Integer

    Time that this measurement occurred Uses epoch minute (epoch time divided by 60)

  • Query: earliest_measurement_time Integer

    Earliest measurement time

  • Query: latest_measurement_time Integer

    Latest measurement time

  • Query: created_at String

    When the record was first created. Use ISO 8601 datetime format

  • Query: updated_at String

    When the record was last updated. Use ISO 8601 datetime format

  • Query: limit Integer

    The LIMIT is used to limit the number of results returned. So if you have 1000 results, but only want to the first 10, you would set this to 10 and offset to 0. The maximum limit is 200 records.

  • Query: offset Integer

    OFFSET says to skip that many rows before beginning to return rows to the client. OFFSET 0 is the same as omitting the OFFSET clause. If both OFFSET and LIMIT appear, then OFFSET rows are skipped before starting to count the LIMIT rows that are returned.

  • Query: sort String

    Sort by given field. If the field is prefixed with '-', it will sort in descending order.
