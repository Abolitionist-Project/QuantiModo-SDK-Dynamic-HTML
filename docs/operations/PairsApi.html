This is the API

pairsGet

Pairs cause measurements with effect measurements grouped over the duration of action after the onset delay.

URL

https://localhost/api/pairs

HTTP Method

GET

Response Type

Parameters

  • Query: cause String

    Original variable name for the explanatory or independent variable

  • Query: causeSource String

    Name of data source that the cause measurements should come from

  • Query: causeUnit String

    Abbreviated name for the unit cause measurements to be returned in

  • Query: delay String

    Delay before onset of action (in seconds) from the cause variable settings.

  • Query: duration String

    Duration of action (in seconds) from the cause variable settings.

  • Query: effect String

    Original variable name for the outcome or dependent variable

  • Query: effectSource String

    Name of data source that the effectmeasurements should come from

  • Query: effectUnit String

    Abbreviated name for the unit effect measurements to be returned in

  • Query: endTime String

    The most recent date (in epoch time) for which we should return measurements

  • Query: startTime String

    The earliest date (in epoch time) for which we should return measurements
