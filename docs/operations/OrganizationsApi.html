This is the API

v1OrganizationsOrganizationIdUsersPost

Get user tokens for existing users, create new users

URL

https://app.quantimo.do/api/v1/organizations/{organizationId}/users

HTTP Method

POST

Response Type

UserTokenSuccessfulResponse

Parameters

  • Path: organizationId Integer

    Organization ID

  • Body: body UserTokenRequest(UserTokenRequest)

    Provides organization token and user ID

  • Query: access_token String

    User's OAuth2 access token
