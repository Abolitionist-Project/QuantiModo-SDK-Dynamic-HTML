This is the API

connectorsGet

A connector pulls data from other data providers using their API or a screenscraper. Returns a list of all available connectors and information about them such as their id, name, whether the user has provided access, logo url, connection instructions, and the update history.

URL

https://app.quantimo.do/api/v2/connectors

HTTP Method

GET

Response Type

inline_response_200_15

Parameters

  • Query: access_token String

    User's OAuth2 access token

  • Query: name String

    Lowercase system name for the data source

  • Query: display_name String

    Pretty display name for the data source

  • Query: image String

    URL to the image of the connector logo

  • Query: get_it_url String

    URL to a site where one can get this device or application

  • Query: short_description String

    Short description of the service (such as the categories it tracks)

  • Query: long_description String

    Longer paragraph description of the data provider

  • Query: enabled Boolean

    Set to 1 if the connector should be returned when listing connectors

  • Query: oauth Boolean

    Set to 1 if the connector uses OAuth authentication as opposed to username/password

  • Query: limit Integer

    The LIMIT is used to limit the number of results returned. So if you have 1000 results, but only want to the first 10, you would set this to 10 and offset to 0. The maximum limit is 200 records.

  • Query: offset Integer

    OFFSET says to skip that many rows before beginning to return rows to the client. OFFSET 0 is the same as omitting the OFFSET clause. If both OFFSET and LIMIT appear, then OFFSET rows are skipped before starting to count the LIMIT rows that are returned.

  • Query: sort String

    Sort by given field. If the field is prefixed with '-', it will sort in descending order.

connectorsIdGet

Returns information about the connector such as the connector id, whether or not is connected for this user (i.e. we have a token or credentials), and its update history for the user.

URL

https://app.quantimo.do/api/v2/connectors/{id}

HTTP Method

GET

Response Type

inline_response_200_16

Parameters

  • Path: id Integer

    id of Connector

  • Query: access_token String

    User's OAuth2 access token
